gitlab: 502 503 err opening storage failed: block dir: /var/opt/gitlab/prometheus/data 解决方法

  • https://blog.csdn.net/yang_quan_yang/article/details/106790568

gitlab 服务器断电：

sudo gitlab-ctl tail 查看log如下：

报错

evel=error ts=2020-06-16T08:07:29.721643025Z caller=main.go:717 err="opening storage failed: block dir: \"/var/opt/gitlab/prometheus/data/01EA0J4D3PDXTX887PN9G65ZY6\": invalcharacter 'e' looking for beginning of value"

解决方法：  

cd /var/opt/gitlab/

sudo systemctl stop gitlab-runsvdir

sudo gitlab-ctl stop

sudo rm-rf var/opt/gitlab/prometheus/data

sudo rm /var/opt/gitlab/gitaly/gitaly.pid

sudo rm ./redis/dump.rdb -rf

sudo systemctl restart gitlab-runsvdir

sudo gitlab-ctl restart
