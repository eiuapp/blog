webdav with docker

bytemark/webdav

https://github.com/BytemarkHosting/docker-webdav 

docker run --restart always -v /home/ubuntu/webdav/bytemark/dav:/var/lib/dav \
     -e AUTH_TYPE=Digest -e USERNAME=your_username -e PASSWORD=your_password \
     --publish 8080:80 -d bytemark/webdav

启动后,把端口打开,也就是成功了.
