根据报错信息，查看`/node\_modules/gitbook-plugin-prism/index.js`文件及gitbook-plugin-prism插件配置可以知道，插件默认不支持以`shell`语法前缀作为`bash`语法前缀的别名，所以会报错，但是支持自定义，添加上以下配置即可： 

"pluginsConfig": {
    "prism": {
        "lang": {
            "shell": "bash"
        }
    }
}
