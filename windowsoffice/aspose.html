使用FileConverter转换office套件为pdf等格式

  • 百度网盘(或其它地方)中找到 FileConverter 相关文件
  • 百度网盘(或其它地方)中找到 aspose-example 相关文件
# help
./bin/FileConverter -h
./bin/FileConverter -v
# docx 转 pdf
./bin/FileConverter -i ./input.docx -o output.pdf
./bin/FileConverter -i /home/a/Downloads/FileConverter-1.0-SNAPSHOT/input.docx -o output1.pdf
# 加水印(水印是一张图片，或文字(涉及字体)) 注意：加水印时，输出时，必须同时指定，带水印和不带水印的输出文件
./bin/FileConverter -i ./input.docx -w ./example/watermark.pdf -wo output2-water.pdf -o output3.pdf

水印方式相当与在原文件之上贴膜。 当水印为100%时，则只显示水印，不显示文件。 程序jar打出的水印的透明度是 40%.
