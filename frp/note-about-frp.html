    • 安全地暴露内网服务

      如果报了下面的错误，怎么办？ 

      ➜  frp_0.23.3_linux_amd64 git:(master) ✗ ./frpc -c ./frpc.ini
zsh: exec format error: ./frpc
➜  frp_0.23.3_linux_amd64 git:(master)

      哈哈，是因为下载的包，不对，因为我们是 Mac ,不是 linux 

      ➜  frp git:(master) ✗ wget https://github.com/fatedier/frp/releases/download/v0.23.3/frp_0.23.3_darwin_amd64.tar.gz

      最后 mac 的配置如下： 

      ➜  frp_0.23.3_darwin_amd64 git:(master) ✗ cat frpc.ini
# frpc.ini
[common]
server_addr = XXXXXXXX
server_port = XXXXXXXX

[secret_ssh_visitor]
type = stcp
# stcp 的访问者
role = visitor
# 要访问的 stcp 代理的名字
server_name = secret_ssh
sk = XXXXXXXX
# 绑定本地端口用于访问 ssh 服务
bind_addr = 127.0.0.1
bind_port = 6000
➜  frp_0.23.3_darwin_amd64 git:(master) ✗

      登陆一下
