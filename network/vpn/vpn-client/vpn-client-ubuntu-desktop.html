ubuntu desktop connect vpn server

vpn Identity

settings => network => vpn => + => select "Layer 2 Tunneling Protocol(L2TP)"

写入： 

name: 随便写

Gateway: 18.162.123.234 (your vpn server ip address)

username: vpn03
password: your-password

NT Domain: 空

IPsec Settings

click "IPsec Settings", show "IPsec tunnel to L2TP host" window, 勾选 "Enable IPsec tunnel to L2TP host"(前面打勾),

General 下

Gateway ID 为空, Pre-shared key 写入共享密码，比如服务端设置共享密码是"vpn", 那这里就写上"vpn"

Advanced 下

Phase1 Algorithms: 写 3des-sha1-modp 1024 Phase2 Algorithms: 写 3des-sha1

PPP Settings

click "PPP Settings", show "L2TP PPP Options" window,

Authentication 下

勾选(前面打勾), 

PAP, CHAP, MSCHAP, MSCHAPv2, EAP

Security and Compression 下

勾选(前面打勾), 

Allow BSD data compression
Allow Deflate data compression
Use TCP header compression
Use protocol field compression negotiation
Use Address/Control compression
