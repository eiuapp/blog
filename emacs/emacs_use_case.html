how to add number to the end of list

case 1

question

把 

JOE
JOE
JOE
JOE

变成： 

JOE2
JOE3
JOE4
JOE5

answer

设置一个临时变量　count，让它从1开始，那么下一个就是2了。如果希望下一个是10，那这里就把临时变量设置成9. 

M-x eval-expression (setq count 1)

选择 JOE 区域 

M-x set-mark-command 或者 C-SPC

使用正则表达式，其实就是替换掉行尾符 

M-x query-replace-regexp $ RTN \,(incf count) RTN ! RTN
