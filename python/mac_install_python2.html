mac 安装 python2

  • https://pythonguidecn.readthedocs.io/zh/latest/starting/install/osx.html
brew install python@2

看一下，上面的最后输出，有显示 python2 的安装路径，比如，我的就是： /usr/local/Cellar/python@2/2.7.17_1

因为 python@2 像一个“桶”，我们需要再次更新我们的``PATH``，以指向我们的新安装： 

ls /usr/local/Cellar/python@2/2.7.17_1/bin
# export PATH="/usr/local/opt/python@2/libexec/bin:$PATH"
# export PATH="$PATH:/usr/local/Cellar/python@2/2.7.17_1/bin"
export PATH="/usr/local/Cellar/python@2/2.7.17_1/bin:$PATH"

Homebrew命名可执行文件 python2，以便您仍然可以通过可执行文件“python”运行系统Python。
