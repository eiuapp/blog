navicat的破解注册机出现错误提示No All Pattern Found! File Already Patched?

  • https://www.cnblogs.com/sunbr/p/12547548.html

第一步：先把注册机放入安装目录。 （这一步非常关键，先不要打开桌面上安装好的快捷方式！！）

第二步(可先尝试跳过，我没有操作这一步)：如果之前下载过，把注册表清理干净

　　计算机\HKEY_CURRENT_USER\SOFTWARE\PremiumSoft（把这个目录下面的东西全部删了！）

第三步：右键管理员身份运行注册机程序，点击“path”按钮 　　这时会重新生成一个新的文件navicat.exe

第四步：双击运行新的navicat.exe，点击“Generate”获取许可证： 　　再激活，将请求码复制放入点击左下方的“Generate”，成功出现激活码！

安装包和注册机下载地址：https://www.lanzous.com/iaj8tsj
