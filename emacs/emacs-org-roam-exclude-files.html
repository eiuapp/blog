m1: Having More Than One Org-roam Directory

ref

  • https://org-roam.readthedocs.io/en/master/configuration/
  • https://github.com/org-roam/org-roam/issues/394
  • https://github.com/org-roam/org-roam/issues/80

目标

  • 配置文件中的 org-roam-directory 是 "~/install/git/wiki/note"
  • 第二个 org-roam-directory 是 "~/install/git/wiki/emacs"

如何把 "~/install/git/wiki/note" 和 "~/install/git/wiki/emacs" 中的内容，合并到一个 org-roam.db 中，这样在 http://localhost:9090/ 中可以显示关联。

step

  • https://github.com/org-roam/org-roam/issues/394#issuecomment-610123877

.dir-locals.el

在 第二个 org-roam-directory 中创建 .dir-locals.el 文件，内容如下：

``` ((nil . ((eval . (setq-local org-roam-directory (expand-file-name "./")))))) ;; ((nil . ((org-roam-directory . "~/install/git/wiki/emacs")))) ```

C-h v org-roam-directory

然后在 第二个 org-roam-directory 中创建或打开org文件，运行 C-h v org-roam-directory,则 org-roam-directory 的变量值 会显示的是 第二个 org-roam-directory 。 同时 gives me an org-roam.db file in the second directory.

M-x org-roam-find-file

当你运行 M-x org-roam-find-file to create or open file, 这时，会create在 第二个 org-roam-directory 的，这样还是比较方便的。

M-x org-roam-db-build-cache

running M-x org-roam-db-build-cache ， http://localhost:9090/ 中并没有显示 emacs 的图。

但是，怎么样才能让 http://localhost:9090/ 中并没有显示 emacs 的图呢？

(看下文)

``` (setq org-roam-directory (concat wiki-root-dir "/emacs")) ```

emacs C-c C-e, then ~M-x org-roam-db-build-cache~，then open http://localhost:9090/ will show the directory's org roam.

m2: change org-roam-directory in emacs

``` (setq org-roam-directory (concat wiki-root-dir "/notes")) (setq org-roam-directory (concat wiki-root-dir "/emacs")) (setq org-roam-directory (concat wiki-root-dir "/dotnet")) ```

emacs C-c C-e, then ~M-x org-roam-db-build-cache~，then open http://localhost:9090/ will show the directory's org roam.
