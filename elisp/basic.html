Converting Between Symbols and Strings

Sometimes you might need to convert a symbol to string (or vice versa) in Emacs Lisp. You start looking for functions like symbol-to-string and string-to-symbol but, alas, they do not seem exist. Do not despair! Such functions do actually exist, although their names are likely to surprise you: 

(symbol-name 'some-symbol)
; => "some-symbol"
(intern-soft "some-symbol")
; => some-symbol

Emacs Lisp - 声明具有变量/变量名称的函数

转载自 http://www.uwenku.com/question/p-qqsirkgg-ec.html

我正在开发一个Emacs Lisp包，我想添加的一个特定功能是能够在运行中定义函数 - 它们将遵循相同的命名约定，但这将帮助我不必手动声明每一个。Emacs Lisp - 声明具有变量/变量名称的函数

举个例子，我有一个名为Exec中的基本功能，这需要一个参数是可执行推出名称： 

(def exec (cmd)
    (async-shell-command cmd "buffer"))

与此同时，在这种特殊情况下，我知道了我将要使用的可执行文件列表 - 或者更准确地说，我知道如何获取它们的列表，因为它可能随时间而改变。所以我想这样做，因为可执行以下列表的内容： 

("a" "b" "c")

是遍历他们和每一个创建函数的名称exec- [执行] - EXEC-A，exec- b，exec-c。

不幸的是，defun不会评估NAME参数，所以我不能动态地创建函数名。

PS。 exec命令本身已经足够好了 - 它使用completing-read以及提供的可执行文件列表， 但我认为上面的内容会很好。 

(dolist (name name-list)
    (defalias (intern (concat "exec-" name))
    `(lambda() ,(format "Run %s via `exec'." name) (interactive) (exec ,name))))
