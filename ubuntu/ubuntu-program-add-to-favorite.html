添加emacs图标至Favorite

添加图标to applications

$ cat /usr/share/applications/emacs.desktop
[Desktop Entry]
Name = emacs
Comment= emacs
Exec=/usr/local/bin/emacs
Icon=/home/a/Desktop/e.png
Terminal=false
Type=Application

Icon 可以是任意一张可访问到的图片.

参考: https://www.jianshu.com/p/85ac752b13f7

add to favorites

Super+A：显示应用程序菜单, 找到对应程序, 右鍵, Add to Favorites .

把多个版本emacs图标加入到 favorite   favorite

$ cat /usr/share/applications/emacs.desktop
[Desktop Entry]
Name = emacs
Comment= emacs
Exec=/usr/local/bin/emacs
Icon=/home/a/Desktop/e.png
Terminal=false
Type=Application

$ cat /usr/share/applications/emacs270.desktop
[Desktop Entry]
Name = emacs270
Comment= emacs270
Exec=/usr/local/bin/emacs270
Icon=/home/a/Desktop/e270.png
Terminal=false
Type=Application

$

如果，只是在 /usr/share/applications/emacs.desktop 中修改 Exec, 那么，点 favorite 会另外弹出一个emacs窗口，而不是在 favorite 位置出emacs窗口，那么这时，会使用 emacs 的默认图标。 所以，要同时修改 Icon, Name, Comment. 最好的做法是： sudo cp /usr/share/applications/emacs.desktop /usr/share/applications/emacs270.desktop 然后，修改成上面的样子。 尝试一下，会达到之前的效果。然后再删除 favorite 和 /usr/share/applications/emacs.desktop
