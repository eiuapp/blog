egg with postgresql

sequelize queries returning only strings

当数据库是 postgresql 时，为什么，取到的数据类型是 string, 而不是 int ？

  • https://github.com/sequelize/sequelize/issues/2383#issuecomment-58006083
  • https://github.com/brianc/node-postgres/pull/353

https://github.com/brianc/node-postgres/pull/353#issuecomment-283709264 

var pg = require('pg');

// bigint
pg.types.setTypeParser(20, function (value) {
    return parseInt(value);
});

// numeric
pg.types.setTypeParser(1700, function (value) {
    return parseFloat(value);
});

so, need add above code snippet into app.js. 

'use strict';
const start = Date.now();
module.exports = app => {
  app.logger.info('启动耗时 %d ms', Date.now() - start);
  app.beforeStart(async () => {
    const ctx = app.createAnonymousContext();
    // 初始化延迟任务
    app.initDelayTask();

    var pg = require('pg');
    // bigint
    pg.types.setTypeParser(20, function (value) {
        return parseInt(value);
    });
    // numeric
    pg.types.setTypeParser(1700, function (value) {
        return parseFloat(value);
    });
  });
  app.logger.info('process.env.NODE_OPTIONS is: ' + process.env.NODE_OPTIONS);
};

It's effective.

another method is: add a new data type for sequelize. how to add new data type, reference here.

send email

egg上发送邮件

egg-full-email

use the [egg-full-email][https://github.com/alex-my/egg-full-email/]

requirement

QQ邮箱开通 POP3/SMTP 服务. 开通方式： https://service.mail.qq.com/cgi-bin/help?subtype=1&&id=28&&no=1001256

configuration

参考文档的配置就行了 

fullEmail: {
  service: 'QQ', // https://nodemailer.com/smtp/well-known/, eg: qiye.aliyun, 126, QQ（而不是qq）
  port: '465', // SMTP port
  ssl: true, // use ssl
  user: '766529074@qq.com',
  password: 'qq-POP3/SMTP-授权码-不是QQ本身的密码', // POP3/SMTP服务授权码
  from: '766529074@qq.com', // eg: egg-full-email <alex_my@126.com>
  debug: false,
},

然后，在调用的函数中写入 to, subject, html, 

async resetPassword() {
  // After changed the password
  const { ctx, app } = this;
  ctx.logger.info(`start resetPassword!`);

  const to = '273412935@qq.com';
  const subject = 'reset password';
  const html = '<p><span style="color: red">You have now changed your password. </span></p>';

  // result: true if success, or false
  const result = await app.fullEmail.sender(to, subject, html);
  ctx.logger.info(`result: ${result}`);

  this.ctx.body = 'reset password success';
}
