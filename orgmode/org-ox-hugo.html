The HUGO_SECTION is the bare-minimum requirement to specify the destination path.

所以，有3个原则：

  • 当一个org file, 对应于一个文件夹中的一个markdown文件，只设置 #+HUGO_SECTION:, 且使用 C-c C-e H h, 不要去使用 subtree :EXPORT_HUGO_SECTION:. subtree 可以没有 PROPERTIES.
  • 当一个org file, 对应于一个文件夹中的多markdown文件时，只设置 #+HUGO_SECTION:, 不要去使用 subtree :EXPORT_HUGO_SECTION:~。subtree 必有 PROPERTIES。必须要使用　:EXPORT_FILE_NAME:~　以指定不同的文件名。未指定~:EXPORT_FILE_NAME:~的subtree，不会被转换。
  • 当一个org file, 要转换至多个文件夹的多个markdown文件时，请（且只能）使用　subtree :EXPORT_HUGO_SECTION:~，且可以方便使用 ~C-c C-e H A 一次性更新所有的文件夹中的多个markdown文件。如果只更新一个subtree,则必须到对应subtree下，使用 ~C-c C-e H H~。

所以，如果想　subtree 绕开　PROPERTIES, 那就只能是一个 org file　对应一个 markdown文件。

至于，你想使用上面３个中的哪一个原则，示具体情况而定吧。

marking-files-to-not-be-exported

  • https://ox-hugo.scripter.co/doc/tags-and-categories/#marking-files-to-not-be-exported

这里提到了 add that special tag to the org-export-exclude-tags variable. but how to do that? 然后，我　~C-h v org-export-exclude-tags~ 查到已有一个值 noexport, 所以，可以直接在 subtree 中使用这个 tag. 但是，如果你就是喜欢用自己的一个特定tag来表示"do not export"的含义，那么，仔细看~C-h v org-export-exclude-tags~的输出， 最后一行，有提到，~This option can also be set with the EXCLUDE_TAGS keyword.~, 也就是说，我们可以在org file中使用 #+:EXCLUDE_TAGS: abc 的方式，使得所有在subtree中含有~abc~的tag会被忽略，不被oxport.但是，请注意，此时，在这个org file中， org-export-exclude-tags~的默认值~noexport~会失效。 所以，这里有一个权衡，如果你使用了~#+:EXCLUDE_TAGS: abc, 则 :noexport: 无法生效；如果使用 :noexport: ,则不能使用 #+:EXCLUDE_TAGS: abc. 我个人推荐使用 :noexport:,　因为既然是二选一，为何不用默认值呢。而且~#+:EXCLUDE_TAGS: abc~，还要去注意tag值~abc~的编写是否正确之类的问题。

HUGO_TAGS vs filetags

subtree 中的使用原则：

  • HUGO_TAGS不空, filetags不空: 则　~C-c C-e H A~ 输出只有 HUGO_TAGS的 tags
  • HUGO_TAGS为空，filetags不空：则　~C-c C-e H A~ 输出，正常，是　filetags中的tags 和 subtree headline 的 tags
  • HUGO_TAGS为空，filetags为空：则　~C-c C-e H A~ 输出，正常，是　subtree headline 的 tags

HUGO_PREFER_HYPHEN_IN_TAGS

#+filetags: filetags "filetagsa" 'filetagsb' filetagsc_filetagsd filetagse__f filetagesg___h

will transform to markdown 

tags = ["filetags", "filetagsa", "'filetagsb'", "filetagsc-filetagsd", "filetagse__f", "filetagesg_h"]

注意观察上面的转换。

  • 无引号，转换后，双引号
  • 双引号，转换后，双引号
  • 单引号，转换后，单引号
  • _号，转换后，-号
  • __号, 转换后，__号
  • ___号，转换后，_号
