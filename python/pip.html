# pip install -i https://pypi.doubanio.com/simple/ --trusted-host pypi.doubanio.com your_package_name
pip install -i https://pypi.doubanio.com/simple/ --trusted-host pypi.doubanio.com pyinstaller

写在配置文件里

  • https://www.cnblogs.com/zhangruoxu/p/6370107.html

linux/mac用户将它命名为pip.conf, windows用户将它命名为pip.ini. 文件中写如下内容: 

[global]
timeout = 60
index-url = https://pypi.doubanio.com/simple

~注意~： 如果使用http链接，需要指定trusted-host参数 

[global]
timeout = 60
index-url = http://pypi.douban.com/simple
trusted-host = pypi.douban.com

将该文件放置在指定位置.

linux下指定位置为 $HOME/.config/pip/pip.conf 或者 $HOME/.pip/pip.conf

mac下指定位置为 $HOME/Library/Application Support/pip/pip.conf 或者 $HOME/.pip/pip.conf

windows下指定位置为 %APPDATA%\pip\pip.ini 或者 %HOME%\pip\pip.ini

Requirement already satisfied: pyinstaller in /home/a/.local/lib/python3.6/site-packages (3.5)
Requirement already satisfied: altgraph in /home/a/.local/lib/python3.6/site-packages (from pyinstaller) (0.16.1)
Requirement already satisfied: setuptools in /usr/lib/python3/dist-packages (from pyinstaller) (39.0.1)

https://stackoverflow.com/questions/45345377/python-module-not-found-even-though-requirement-already-satisfied-in-pip

然后，你会发现 home/a.local/bin 下会有 pyinstaller, 所以只要把 这个路径加入 PATH 就可以直接使用 pyinstaller 了。
