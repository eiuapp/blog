Connecting to a remote host using multiple hops

m1

m1.1 add-to-list 'tramp-default-proxies-alist

  • https://stackoverflow.com/questions/715855/how-can-i-use-emacs-tramp-to-double-hop-ssh
  • https://www.gnu.org/software/tramp/#Multi_002dhops
(add-to-list 'tramp-default-proxies-alist
                 '("HOSTB" nil "/ssh:USERA@HOSTA:"))

Where HOSTB is the destination host behind HOSTA.

Then type /ssh:USERB@HOSTB: and emacs will prompt for the HOSTA password then the HOSTB password.

m1.2 add-to-list 'tramp-default-proxies-alist

  • https://stackoverflow.com/questions/5847814/emacs-tramp-ssh-double-hop

    I want to work on machine2.abc.def.edu to which I can connect only through machine1.abc.def.edu. My username is myname, on both machines same. 

(add-to-list 'tramp-default-proxies-alist
             '("\\`machine2\\'"
               nil
               "/ssh:%u@machine1.abc.def.edu:"))

Then I do: C-x C-f /ssh:machine2.abc.def.edu or: C-x C-f /ssh:rsuhada@machine2.abc.def.edu
