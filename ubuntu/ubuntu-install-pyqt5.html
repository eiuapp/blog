Ubuntu18.04 安装 PyQt5

https://zhuanlan.zhihu.com/p/137571552

安装pip3

没有对应的pip3，于是执行命令 sudo apt-get install python3-pip ，即可成功安装pip3

首先更新pip3：

sudo python3 -m pip install –upgrade pip

安装 PyQtWebEngine==5.12.1

  • https://stackoverflow.com/questions/55812117/pyqt-qtwebenginewidgets-with-pyqt-5-12

因为，后续的包加载过程中，很可能会有导入错误，所以，要选择相互匹配的版本。目前（2020.11.11）最好选择 5.12 版本。

sudo pip install PyQtWebEngine==5.12.1 -i https://pypi.doubanio.com/simple/ –trusted-host pypi.doubanio.com

安装 pyqt5==5.12.0

sudo pip3 install pyqt5==5.12.0

如果pyqt5后面没有指定版本，将安装最新版本5.14.1，但是这个版本太新，出现兼容性问题，缺少libqsqlmysql.so，不支持连接mysql数据库，而5.12.0则可支持mysql。 5.14.1支持odbc，sqlite，psql；5.12.0可支持sqlite，mysql，psql。故放弃5.14采用5.12。

安装图形设计工具

sudo apt install pyqt5-dev pyqt5-dev-tools *安装pyqt5的依赖项 sudo apt install qt5-default qttools5-dev-tools * 安装qtdesigner
