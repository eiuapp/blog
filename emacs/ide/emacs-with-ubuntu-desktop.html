DONE snappet

先要把 snippets 从git中取出, 并放入 ~/.spacemacs.d/snippets/

让~/.spacemacs.d/snippets 生效。

输入 sh 然后，会有提示，C-j 或者 C-k 选中，再按 RTN或C-RTN 就可以成功使用了。

org-mode下 TAB 不能展开或缩进

通过 SPC h d k 按下 TAB, 得到下面的内容

``` TAB runs the command evil-jump-forward (found in evil-motion-state-map), which is an interactive compiled Lisp function in ‘evil-commands.el’.

It is bound to <C-i>, TAB.

(evil-jump-forward &optional COUNT)

Go to newer position in jump list. To go the other way, press C-o. ```

快捷键 命令 说明 S-TAB org-shifttab 循环切换整个文档的大纲状态（三种状态：折叠，打开下一级，打开全部） TAB org-cycle 循环切换光标所在大纲的状态

所以，真正的命令是 org-cycle .

但是现在是点击标题后，在 insert mode 下可以按TAB成功展开，但是normal模式下，不行呀！～不知道为什么
