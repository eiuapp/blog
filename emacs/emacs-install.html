install 26.3 Dependencies

http://ergoemacs.org/emacs/building_emacs_on_linux.html 

sudo apt-get install build-essential
sudo apt-get build-dep emacs25

update emacs version from 26.1 to 26.3

download address is :

  • https://www.gnu.org/software/emacs/download.html
  • https://github.com/emacs-mirror/emacs/releases
wget -O ~/Downloads/emacs-26.3.tar.xz http://mirrors.nju.edu.cn/gnu/emacs/emacs-26.3.tar.xz
sudo apt-get build-dep emacs
tar xf emacs-26.3.tar.xz
cd emacs-26.3  # 进入安装目录
./configure
make
sudo make install
which emacs
emacs --version

REF

  • https://www.linuxidc.com/Linux/2019-09/160451.htm

安装 26.3 后，重启了电脑，报错误

没重启电脑的情况下，可以使用. 安装 26.3 后，重启了电脑，就报下面这个错误 

Eager macro-expansion failure: (error "File ‘/usr/local/share/emacs/26.3/lisp/emacs-lisp/advice.elc’ was not compiled in Emacs")
Eager macro-expansion failure: (error "File ‘/usr/local/share/emacs/26.3/lisp/emacs-lisp/easy-mmode.elc’ was not compiled in Emacs") [2 times]
define-minor-mode: File ‘/usr/local/share/emacs/26.3/lisp/emacs-lisp/debug.elc’ was not compiled in Emacs
mwheel-scroll: End of buffer [4 times]

重新再编译,安装,在没有重启电脑的情况下,继续使用着. (等下次重启完电脑再来记录情况)

configure: error: no acceptable C compiler found in $PATH

就是 gcc not found 找到 https://www.linuxidc.com/Linux/2019-06/159059.htm， 发现没有必要。我直接回到

http://ergoemacs.org/emacs/building_emacs_on_linux.html 

sudo apt-get install build-essential
sudo apt-get build-dep emacs25

就可以了。
