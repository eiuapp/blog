Remote-ssh服务器端vscode-server安装失败

  • https://blog.csdn.net/wyg1997/article/details/101460961

getPlatformForHost was canceled

[12:23:33.295] Running script with connection command: ssh -T -D 34469 -o ConnectTimeout=15 tlservera bash [12:23:33.473] > 914371f7c00d: running [12:23:33.474] Got some output, clearing connection timeout [12:23:33.490] > Acquiring lock on home/ubuntu.vscode-server/bin/a5d1cc28bb5da32ec67e86cc50f84c67cc690321/vscode-remote-lock.ubuntu.a5d1cc28bb5da32ec67e86cc50f84c67cc690321 > ln home/ubuntu.vscode-server/bin/a5d1cc28bb5da32ec67e86cc50f84c67cc690321/vscode-remote-lock.ubuntu.a5d1cc28bb5da32ec67e86cc50f84c67cc690321.target home/ubuntu.vscode-server/bin/a5d1cc28bb5da32ec67e86cc50f84c67cc690321/vscode-remote-lock.ubuntu.a5d1cc28bb5da32ec67e86cc50f84c67cc690321

去对应的　/home/ubuntu/.vscode-server/bin/a5d1cc28bb5da32ec67e86cc50f84c67cc690321/　文件夹中，把　快捷方式　删除再重启vscode 

ubuntu@utuntu:~/.vscode-server/bin/a5d1cc28bb5da32ec67e86cc50f84c67cc690321$ rm vscode-remote-lock.ubuntu.a5d1cc28bb5da32ec67e86cc50f84c67cc690321

杀掉VSCODE server 重开即可

$ pgrep -f "vscode" | xargs kill
